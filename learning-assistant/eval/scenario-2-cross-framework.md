# Eval: 场景2 — 有经验者跨框架

## 学习者设定

3年 Vue 经验 + TypeScript，想学 React 18+。每周 10 小时。

## 教练指令

按 `SKILL.md` 执行完整 Phase 1→2→3 流程。
重点验证：JS→Python 映射、不解释基础概念、练习从 L2/L3 起步。

## 期望行为

1. 识别 Vue 经验是可迁移知识，做精确概念映射（`useEffect` cleanup ≈ `onUnmounted`）
2. 绝不解释基础编程概念（变量、循环、函数、if/else）
3. 练习从 L2 起步（补全代码），非 L1（照抄修改）
4. 项目设计有真实业务场景（表单提交+异步验证），非 toy project

## 评测标准（3 维度）

### 1. 规则执行率
- [ ] A层: 费曼检验 (必须)
- [ ] A层: 讲练同步 (必须)
- [ ] A层: 跨技术概念映射 (必须)
- [ ] B层: 主动回忆 (会话开头问"上次学了什么")
- [ ] 不解释基础概念 (WF9)

### 2. 学习者体验
- [ ] 有没有被当作"初学者"对待的冒犯感？
- [ ] 概念映射是否精确到 API 层面？
- [ ] 练习难度是否匹配 3 年经验水平？

### 3. 学习有效性
- [ ] 费曼检验通过了吗？
- [ ] 学习者能独立完成 L2/L3 练习吗？
- [ ] 学习者对 React 和 Vue 的关键差异能用自己的话解释吗？

## 评测方式

用 Agent 运行此场景：1个学习者 Agent + 1个教练 Agent 对话 8-10 个来回。
对话结束后，评测 Agent 按上述标准打分并输出报告。
